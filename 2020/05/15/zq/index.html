  • events

    • eventBus适合用于小型项目，大型项目建议用vuex

    • 子组件向父组件传值

      父组件通过$on事件监听当前实例上的自定义事件，而其子组件可以通过$emit触发该自定义事件。示例如下：

      父组件：

      <div @test="handleTest">测试test方法,在子组件中触发</div>
// 根据自身实际情况确定在哪儿触发
<child @click="invokeEmit">点击我触发test事件</child>
...
methods: {
    handleTest(v) {
        console.log(v)
    }
}
...

      子组件child.vue：

      ...
methods: {
    involeEmit() {
        // 向父组件发送自定义事件 第一个参数是自定义事件的名称，必须跟父组件定义的名称一致
        // 第二个参数是可以携带给父组件的值
        this.$emit('test','我是测试数据')
    }
}
